Frequently asked

What is OpenClaw?

OpenClaw is an open-source AI assistant framework with 270k+ GitHub stars, supporting 25+ messaging channels. SparkClaw hosts it for you.

Can I bring my own LLM API keys?

In V0, instances use our managed LLM gateway (Prism). BYOK support is on the roadmap.

What happens if I cancel?

Your instance is suspended but your data is kept for 30 days. Re-subscribe to restore it.

Which LLM models are supported?

GPT-4o, Claude, Qwen 2.5, and more via our Prism gateway. Model selection available in setup.

Is my data private?

Each user gets a fully isolated container. Your data is never shared with other users.

Telegram Bot Setup 
1. Open @BotFather on Telegram
2. Send /newbot and follow prompts
3. Copy the bot token
4. Paste it in your OpenClaw setup page
5. Done! Your bot is live.
